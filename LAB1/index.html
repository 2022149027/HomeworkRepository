인터넷프로그래밍 수업을 신청한 이유

인터넷프로그래밍 수업에서 HTML과 CSS를 배우는 것은 기본이고, 이번 기회에 JAVASCRIPT
와 백엔드 분야인 node.js, 그리고 기초적인 SQL 데이터베이스 문법 등 웹에 사용되는 최신
프로그래밍 기법들을 공부하면서, 웹 서비스를 스스로 구축할 수 있는 능력을 기를 수 있을
것 같아서 인터넷프로그래밍 수업을 신청했습니다. 이번 기회를 통해 팀 프로젝트를 하면서
창의력, 구현 능력과 협업 능력을 향상시킬 수 있다고 생각합니다.
